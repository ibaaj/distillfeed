{% if is_arxiv_scope %}How the daily arXiv digest works{% else %}How this summary is updated{% endif %}

{% if is_arxiv_scope %}

Check arXiv reads every configured category and locally screens unseen papers. It does not call the ranking model.

Update daily digest checks first, then—only when a new announcement day is waiting—ranks the shortlist and publishes one combined digest. Repeating it on the same announcement does not create another digest.

This specialist recipe, shortlist and ntfy threshold are separate from ordinary feed summaries.

Open arXiv digest settings{% else %}

Check feeds reads the selected RSS and Atom documents and stores new feed entries. It does not open linked webpages or use AI.

Update this summary checks those feeds, then sends waiting titles and feed-provided descriptions to your configured model. Each entry is evaluated once after a successful response; DistillFeed then publishes one new summary for this exact selection.

Entries excluded by you are never selected. Sources set to “Only on request” are included when you update their own summary.
